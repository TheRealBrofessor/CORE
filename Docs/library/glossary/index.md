# Glossary

## A
- **ACL (Access Control List)** — rules defining who/what can access a resource.
- **ARP (Address Resolution Protocol)** — maps IP addresses to MAC addresses on a LAN.

## C
- **CIDR** — notation for IP ranges (e.g., `10.0.0.0/24`).

## D
- **DNS** — resolves names to IP addresses.

## E
- **EUID/EGID** — effective user/group IDs used for permission checks.

## F
- **Firewall** — rules that allow/deny network traffic based on policy.

## I
- **IAM** — identity and access management (users/roles/policies).
- **ICMP** — control messages used by tools like `ping`.

## P
- **Port** — logical endpoint for a network service (0–65535).
- **Process** — running program instance with PID.

## S
- **SSH** — secure remote shell protocol.
- **SUID** — permission bit that runs a binary as the file owner.

## T
- **TLS** — cryptographic protocol behind HTTPS.
